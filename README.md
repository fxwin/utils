# utils
Collection of scripts/snippets/configs that I use:
- [uv-sync](githooks/uv-sync): git hooks that keep my uv env in sync when checking out branches/commits
- [verify-ticket-nr](githooks/verify-ticket-nr): git hook that forces commit messages to start with an ID of the form `#1234`. Useful when working with tools that automatically link commits to tickets such as Azure DevOps
- [llmpeg](llmpeg/): crop/edit videos using natural language. Output can be piped into [share](vps-share/)
- [share](vps-share/): easily share videos (Or other files) via a VPS
